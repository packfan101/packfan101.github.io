Typography Plan

Headings

Headings will all be in the same font.

  • Font Type: Shippori Mincho from Google Fonts
  • Font Weight: 800
  • Backup Font: Georgia
  • Tertiary Font: Serif

Paragraphs/Footer

All other text on the site will fall into the following font.
The default font size will be 16px for the page.

  • Font Type: Roboto from Google Fonts
  • Font Weight: 400 Regular
  • Backup Font: Veranda
  • Tertiary Font: Sans-Serif

Hyperlinks

Hyperlinks will follow the same font style as the headings. The font color will match the color scheme.

  • Font Type: Shippori Mincho from Google Fonts
  • Font Weight: 800
  • Backup Font: Georgia
  • Tertiary Font: Serif
